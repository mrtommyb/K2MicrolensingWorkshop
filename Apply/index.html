Application and Registration

To apply to attend the workshop either in-person on virtually you will need to fill in the application form. The deadline for applying to attend is April 17, 2015 at 4.00pm PST. This is a hard deadline because we need to process badges for all visitors. We would appreciate early registration by any foreign nationals that wish to attend owing to the increased overhead on processing badges.
